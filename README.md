# Hand Gesture Controlled Virtual Mouse 🖐️🖱️

## Project Overview
This project allows users to control their computer mouse using hand gestures captured through a webcam.

The system detects hand gestures in real time and converts them into mouse actions like cursor movement, clicking, scrolling, and screenshot capture.

---

## Features
- Real-time hand tracking
- Cursor movement using hand gestures
- Left click gesture
- Scroll functionality
- Screenshot capture
- Cursor freeze option
